# Portfolio

My portfolio website built with Next.js and React.

## Setup
1. Install dependencies:
   ```sh
   pnpm install
   ```
2. Start the development server:
   ```sh
   pnpm dev
   ```
   
Or visit https://mangoapi.dev to see the live version.
