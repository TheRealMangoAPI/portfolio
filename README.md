This project showcases my 3D portfolio built using Three.js, Three.js-Fiber, and Next.js. It's a result of hands-on learning and exploration following a tutorial from [JavaScript Mastery](https://www.youtube.com/@javascriptmastery).

## About

This portfolio is a visual representation of my development journey as a hobbyist, inspired by the tutorial from [JavaScript Mastery](https://www.youtube.com/@javascriptmastery). Through this project, I aimed to dive into the world of 3D web development and create an interactive space to showcase my skills and projects.

## Installation

To run this project locally, follow these steps:

1. Clone this repository.
2. Install dependencies using `yarn`.
3. Start the development server with `yarn dev`.
4. Access the project at `http://localhost:3000`.
